Project Name:-Arduino uno Radar project.
 
description:-The arduino uno radar project is an ultrasonic sensor for detecting the objects.
             The servo motor is used for rotating the sensor and allows arduino board for 
             controlling them.

Components Required:-
               1.Hardware Components:- a) Arduino uno 
                                       b) HC-SR04 ultrasonic sensor
                                       c) servo motor
                                       d) connecting wires
                                       e) jumper cables
                                       f) USB cable
               
               2.Software required:- a) Arduino IDE
                                     b) processing application.
